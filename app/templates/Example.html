Example

Features

  • The required features are:

    • Type of Preeclampsia - Preterm (less than 37weeks ) and All (Any Gestational Age). Previous prgnancies lasting beyond 24 weeks. If answer is yes.
    • GA - Gestational age where all following biomarker are taken (per days). Default is 89.
    • Pappa - Pregnancy associated plasma protein-A. Default is 2.7.
    • Utpi - Mean uterine. Default is 1.65.
    • Map - Mean arterial pressure. Default is 86.5.
    • Plgf - Angiogenic placental growth factor. Default is 35.
    • PLGF Machine - Type of machine used for plgf. Default is Delfia.
    • Interval - Interval from the end of the last of pregnancy (per years). Default is 3.
    • Last GA - Gestational age of previous pregnancy (per weeks). Default is 40.
    • Age - Mother age. Default is 31.
    • HT - Mother height in cm. Default is 164.9.
    • WT - Mother weigh. Default is 67.1
    • Race - Mother race. Default is White.
    • Conception - Conception type. Default is Spontaneous.
    • Smoking - Whether the mother smokes. Default is No.
    • Family history of PE - Whether the grandmother of the baby had PE. Default is No.
    • Chronic hypertension - Whether the mother has Chronich hypertension. Default is No.
    • Diabetes - Whether the mother has diabetes (and type). Default is No.
    • SLE - Whether the mother has Lupus. Default is No.
    • Previous PE - Whether the mother had previous PE. Default is No.
  • Empty features are filled with default values.
