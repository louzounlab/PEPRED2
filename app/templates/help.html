We introduce here a server that predicts TCR-HLA binding


The input of the model is a list of pairs of TCR and HLA, and the output is a number between 0 and 1, that represent the probability of the TCR-HLA pair to bind The format of the input file should look like the example (CSV).

The csv file needs to include the column: tcra, tcrb, va, vb, ja. jb, t_cell_type, mhc. If you don't have some of the information, you may leave the columns empty.

The input file should contain the same columns as in the example file. You can try to use this file to experience with the model.
