Help


The purpose of this website is to predict the risk to develop preeclampsia (PE), according to first-trimester non-corrected prior risk combined with first-trimester biochemical and biophysical markers.

Input is manually inserted by the user into the fields on the 'Home' page. Each marker has a default value, which is determined by the median value of the data used for training the model.
In case the user did not fill out the field, the default value is used. For markers description and default values, see 'Example'.

After filling out the values and pressing the 'Result' button, the estimated prediction and its percentile appear on the screen.
